Lancer & planifier la mise à jour
Instructions pour exécuter le script manuellement ou automatiquement via cron.
ℹ️ La mise à jour automatique nécessite Python 3.8+ sur ton serveur et la clé API Anthropic. Coût estimé : ~0.05€ par exécution.
1. Tester manuellement
# Exécution manuelle du script python3 veille_ia_updater.py # Avec les variables d'environnement ANTHROPIC_API_KEY=sk-ant-xxx \ SMTP_USER=ton@email.fr \ SMTP_PASSWORD=motdepasse \ EMAIL_TO=toi@email.fr \ python3 veille_ia_updater.py
2. Automatiser avec cron (Linux/Mac)
# Ouvrir le crontab crontab -e # Exécution tous les lundis à 8h00 0 8 * * 1 /usr/bin/python3 /chemin/vers/veille_ia_updater.py >> /var/log/veille_ia.log 2>&1 # Avec rsync pour copier news.json sur le serveur web 0 8 * * 1 /usr/bin/python3 /chemin/vers/veille_ia_updater.py && \ rsync -av /chemin/vers/news.json user@serveur.fr:/var/www/html/veille-ia/ >> /var/log/veille_ia.log 2>&1
3. Via GitHub Actions (si hébergé sur GitHub Pages)
# .github/workflows/veille-ia.yml name: Veille IA Hebdomadaire on: schedule: - cron: '0 8 * * 1' workflow_dispatch: # Bouton manuel dans GitHub jobs: update: runs-on: ubuntu-latest steps: - uses: actions/checkout@v4 - name: Run updater env: ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }} SMTP_USER: ${{ secrets.SMTP_USER }} SMTP_PASSWORD: ${{ secrets.SMTP_PASSWORD }} EMAIL_TO: ${{ secrets.EMAIL_TO }} run: pip install anthropic requests beautifulsoup4 && python3 veille_ia_updater.py - uses: stefanzweifel/git-auto-commit-action@v5 with: commit_message: '🗞 Mise à jour automatique veille IA'
✅ La méthode GitHub Actions est recommandée si ton site est hébergé sur GitHub Pages — c'est gratuit, fiable, et tu peux déclencher une mise à jour manuelle en 1 clic.
